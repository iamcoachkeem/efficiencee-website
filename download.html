Installation Guide

Windows Installation

  1. Download the Windows installer (.exe file)
  2. Run the installer and follow the on-screen instructions
  3. Launch Efficiencee from your desktop or start menu
  4. Enter your trial license key when prompted

macOS Installation

  1. Download the macOS installer (.dmg file)
  2. Open the .dmg file and drag the Efficiencee app to your Applications folder
  3. Launch Efficiencee from your Applications folder
  4. Enter your trial license key when prompted

Linux Installation

  1. Download the Linux AppImage file
  2. Make the file executable: chmod +x enhance-tech-linux.AppImage
  3. Run the AppImage: ./enhance-tech-linux.AppImage
  4. Enter your trial license key when prompted
